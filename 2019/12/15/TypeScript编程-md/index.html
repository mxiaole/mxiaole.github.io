TypeScript编程

Posted on 2019-12-15 Edited on 2020-03-05 In 编程语言

本文简单记录TypeScript中的最基本的知识点

typescript开发的基本流程：

  • 创建.ts为后缀的文件
  • 使用tsc命令编译ts文件，生成js代码，然后在浏览器或者Node.js环境运行。

1 变量

1.1 数据类型

基础数据类型

boolean: true, false
number: typescript中的所有的数值都是浮点类型
string: 可以使用单引号或者双引号来表示字符串

复合数据类型

数组：数据类型[]或者Array<数据类型>
元组：[string, number]
枚举：自定义一个枚举类型，然后使用这个枚举类型进行定义变量。enum Color {Red, Green, Blue}
任意类型：any
没有类型：void，void类型的变量只能赋值undefined，null，undefined和null也是类型
never: 永远不存在的值的类型

联合类型

这是ts中特有的类型。

1.2 创建

基本语法：let variable_name: type = variable_value

上述三种定义变量的方式区别：

  • var定义的变量的作用域是全局的，可以在任意的位置被访问
  • let定义的变量的作用域是块作用域
  • const声明的变量的作用域和let是一样的，但是const定义的变量只能在声明的时候赋值，之后不能修改
